## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your PostgreSQL credentials
node src/models/migrate.js    # Create all tables
npm run dev                   # Start backend on port 5000
```

### Frontend
```bash
cd frontend
npm install
npm start                     # Start React app on port 3000
```
